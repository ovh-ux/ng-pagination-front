pagination-front
directive in module ovh-angular-pagination-front

Description

Pagination module for managers. parameters are saved in local storage

Usage

as attribute
<ANY pagination-front
     items="{Array}"
     paginated-items="{Array}"
     [current-page="{Number}"]
     [nb-pages="{Number}"]
     [items-per-page="{Number}"]
     [refresh="{Any}"]
     [page-placeholder="{String}"]
     [item-per-page-placeholder="{String}"]
     [on-page-change="{Function}"]
     [transform-item="{Function}"]
     [on-transform-item-get-promise="{Function}"]
     [on-transform-item-done="{Function}"]
     [on-transform-item-notify="{Function}"]
     [go-to-page="{Number}"]
     [save-name="{String}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
itemsArray

List of items to paginate

paginatedItemsArray

List of items on the currentpage (those to be displayed)

currentPage
(optional)
Number

Current page number

(default: 1)

nbPages
(optional)
Number

Number of pages

itemsPerPage
(optional)
Number

Number of items per page

(default: 10)

refresh
(optional)
Any

The component is refreshed when this variable change

pagePlaceholder
(optional)
String

Text to display in the page input

itemPerPagePlaceholder
(optional)
String

Text to display in the item per page input

onPageChange
(optional)
Function

Function to execute on change

transformItem
(optional)
Function

Function executed on each item on pagination

onTransformItemGetPromise
(optional)
Function

Function that get the transformItem promise (ie. on-transform-item-get-promise="foo(promise)")

onTransformItemDone
(optional)
Function

Function invoked when transformItem is done on all items

onTransformItemNotify
(optional)
Function

Function invoked when transformItem is done on each item

goToPage
(optional)
Number

Page to reach

saveName
(optional)
String

Name of number value items per page will be saved into localstorage (Optional)
